# 健康評分計算邏輯

總分 100 分，分為 6 個維度：

## 1. 熱量控制 (20分)
```
90-110% 目標：20分
80-90% 或 110-120%：15分
70-80% 或 120-130%：10分
其他：5分
```

## 2. 蛋白質品質 (20分)
```
總量達標：10分
分配良好（≥2餐達亮氨酸門檻）：10分
```

## 3. 脂肪品質 (15分)
```
Omega-3 達 2g：8分
無反式脂肪/少油炸：7分
```

## 4. 碳水品質 (15分)
```
纖維 ≥ 30g：8分
以原型食物為主：7分
```

## 5. 微量營養 (15分)
```
蔬果 ≥ 5份：8分
含發酵食品：4分
植物多樣性 ≥ 10種：3分
```

## 6. 抗發炎 (15分)
```
抗發炎分數 > 0：得分 = min(抗發炎分數 × 3, 15)
```

---

## 評分明細輸出範例
```
[分數] 今日健康評分：78/100

[明細] 評分明細：
┌──────────────┬───────┬──────┐
│ 維度         │ 得分  │ 滿分 │
├──────────────┼───────┼──────┤
│ 熱量控制     │ 18    │ 20   │
│ 蛋白質品質   │ 15    │ 20   │
│ 脂肪品質     │ 12    │ 15   │
│ 碳水品質     │ 10    │ 15   │
│ 微量營養     │ 11    │ 15   │
│ 抗發炎       │ 12    │ 15   │
└──────────────┴───────┴──────┘

[提示] 快速提升建議：
1. 早餐加 2 顆蛋 -> 蛋白質 +8 分
2. 午餐加一份燙青菜 -> 纖維 +3 分
3. 下午喝杯無糖綠茶 -> 抗發炎 +2 分
```
