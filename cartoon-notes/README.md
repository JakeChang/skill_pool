# Cartoon Notes 使用指南

個人動畫卡通觀看紀錄管理系統。記錄每集心得、生成摘要與分析、搜尋網路補充資訊、產生 HTML 報表。

## 安裝

```bash
npx openskills install JakeChang/skill_pool/cartoon-notes
npx openskills sync
```

## 如何觸發此技能

當你的訊息包含以下關鍵字時，會自動啟用此技能：

- 「動畫筆記」、「卡通紀錄」、「追番紀錄」
- 「記錄這部動畫」、「集數心得」、「動畫摘要」
- 「看了 XX 動畫」

## 使用範例

### 新增動畫

```
我要記錄《葬送的芙莉蓮》這部動畫
```

輸出：
```
已建立動畫紀錄：

| 名稱           | 製作公司   | 年份 | 狀態     |
|----------------|-----------|------|----------|
| 葬送的芙莉蓮   | MADHOUSE  | 2023 | watching |

檔案：cartoon-notes/frieren.json
```

### 記錄集數心得

```
記錄芙莉蓮第 1 集：
- 精彩片段：冒險結束後的告別場景
- 心得：開場就展現了時間流逝的沉重感
- 喜愛場景：芙莉蓮看著欣梅爾的雕像
- 經典台詞：「我覺得我對人類的了解還是太少了」
- 評分：5
```

輸出：
```
已記錄第 1 季第 1 集：

集標題：旅途的開始
精彩片段：1 個
心得：已記錄
評分：5/5

是否要搜尋網路補充這集的相關資訊？
```

### 查看追番列表

```
我的動畫列表
```

輸出：
```
你的動畫紀錄：

| 名稱           | 狀態      | 進度      | 評分 |
|----------------|-----------|-----------|------|
| 葬送的芙莉蓮   | watching  | S1 12/28  | -    |
| 鬼滅之刃       | completed | S4 全     | 4.5  |
| 咒術迴戰       | on-hold   | S2 8/23   | -    |
```

### 完成觀看並生成摘要

```
我看完芙莉蓮了，請生成摘要
```

系統會：
1. 彙整所有季/集筆記
2. 生成劇情概要、主題分析、最佳集數
3. 詢問整體評分（1-5）與推薦理由
4. 將狀態設為 `completed`

### 生成 HTML

```
生成動畫紀錄的 HTML 報表
```

輸出：
- `cartoon-notes/reports/index.html` - 動畫列表頁面
- `cartoon-notes/reports/{anime-id}.html` - 各動畫詳情頁面

## 資料儲存

所有紀錄儲存在專案根目錄的 `./cartoon-notes/`：

```
cartoon-notes/
├── frieren.json            # 葬送的芙莉蓮
├── demon-slayer.json       # 鬼滅之刃
├── ...
└── reports/
    ├── index.html          # 動畫列表
    └── frieren.html        # 動畫詳情
```

## 核心功能

- **動畫管理**：新增動畫、追蹤觀看狀態（watching/completed/on-hold/dropped）
- **集數紀錄**：記錄精彩片段、心得、喜愛場景、經典台詞、評分
- **多季支援**：支援多季動畫的分季紀錄
- **網路搜尋**：記錄後可搜尋補充資訊（角色、劇情背景等）
- **摘要生成**：自動彙整分析，產生深度摘要
- **HTML 輸出**：生成美觀的動畫紀錄網頁
