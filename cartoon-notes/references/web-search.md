# 網路搜尋策略

記錄完一集後，主動詢問是否需要搜尋補充資訊。

## 搜尋類型

### 1. 劇情細節
- 關鍵字：`{動畫名} 第{X}季 第{Y}集 劇情`
- 找出遺漏的重要劇情點

### 2. 角色資訊
- 關鍵字：`{動畫名} 角色 {角色名}`
- 補充角色背景、聲優資訊

### 3. 原作對照
- 關鍵字：`{動畫名} 原作 漫畫 對應`
- 動畫與原作的差異

### 4. 彩蛋與伏筆
- 關鍵字：`{動畫名} 第{X}集 彩蛋 伏筆`
- 可能錯過的細節

### 5. 討論與評價
- 關鍵字：`{動畫名} 第{X}集 討論 心得`
- 其他觀眾的觀點

## 執行流程

1. 使用 WebSearch 搜尋
2. 選擇相關度高的結果用 WebFetch 取得內容
3. 整理後以清單呈現
4. 詢問是否加入筆記
