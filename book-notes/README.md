# Book Notes 使用指南

個人閱讀筆記管理系統。記錄書籍章節重點、生成摘要與分析、產生 HTML 報表。

## 安裝

```bash
npx openskills install JakeChang/skill_pool/book-notes
npx openskills sync
```

## 如何觸發此技能

當你的訊息包含以下關鍵字時，會自動啟用此技能：

- 「閱讀筆記」、「讀書筆記」、「書籍筆記」
- 「記錄這本書」、「章節重點」、「書籍摘要」

## 使用範例

### 新增書籍

```
我要開始閱讀「原子習慣」這本書，作者是 James Clear
```

輸出：
```
已建立書籍筆記：

| 書名     | 作者        | 狀態    | 開始日期   |
|----------|-------------|---------|------------|
| 原子習慣 | James Clear | reading | 2026-01-27 |

檔案：data/atomic-habits.json
```

### 記錄章節

```
記錄「原子習慣」第一章：
- 重點：習慣是自我改善的複利
- 重點：1% 的進步每天累積會帶來巨大改變
- 引用：「成功是日常習慣的產物，而非一生僅有一次的巨大轉變。」
- 想法：這讓我反思每天的小決定有多重要
```

輸出：
```
已記錄第 1 章：

章節標題：（可補充）
重點：2 條
引用：1 條
個人想法：已記錄

目前進度：1 章
```

### 完成閱讀並生成摘要

```
我已經讀完「原子習慣」，請生成摘要
```

系統會：
1. 彙整所有章節筆記
2. 生成主要觀點、關鍵收穫、深度分析
3. 詢問你的評分（1-5）與推薦理由
4. 將狀態設為 `completed`

### 查看書籍列表

```
列出我的閱讀筆記
```

輸出：
```
你的書籍列表：

| 書名     | 作者        | 狀態      | 章節數 |
|----------|-------------|-----------|--------|
| 原子習慣 | James Clear | completed | 12     |
| 深度工作 | Cal Newport | reading   | 3      |
```

### 生成 HTML

```
生成閱讀筆記的 HTML 報表
```

輸出：
- `output/index.html` - 書籍列表頁面
- `output/{book-id}.html` - 各書籍詳情頁面

## 資料儲存

所有紀錄儲存在專案根目錄的 `./data/`：

```
data/
├── atomic-habits.json      # 原子習慣
├── deep-work.json          # 深度工作
└── ...

output/
├── index.html              # 書籍列表
├── atomic-habits.html      # 書籍詳情
└── ...
```

## 核心功能

- **書籍管理**：新增書籍、追蹤閱讀狀態
- **章節記錄**：記錄重點、引用、個人想法
- **摘要生成**：自動彙整分析，產生深度摘要
- **評分推薦**：完成後給予評分與推薦理由
- **HTML 輸出**：生成美觀的閱讀筆記網頁
