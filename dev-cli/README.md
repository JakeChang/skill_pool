# dev-cli

Node.js CLI/TUI 應用開發技能，使用 TypeScript + meow 框架，遵循三層式架構（CLI Entry → Core → Utils）。

## 安裝

```bash
npx openskills install JakeChang/skill_pool/dev-cli
npx openskills sync
```

## 指令

| 指令 | 用途 | 產出 |
|------|------|------|
| `/dev-cli plan` | 規劃 CLI 指令結構與功能 | `docs/requirements/{tool}.md` |
| `/dev-cli spec` | 撰寫技術計劃書 | `docs/specs/{tool}.md` |
| `/dev-cli dev` | 依三層架構開發 | 程式碼 |
| `/dev-cli review` | 檢查程式碼品質（架構、CLI、TUI、安全性） | 審查報告 |
| `/dev-cli test` | 終端機實際操作測試 | 測試報告 |
